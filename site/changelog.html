🎲 ROFLFaucet Changelog

Significant updates and improvements to ROFLFaucet

  • 2026-04-12 · ROFLFaucet — Bug Fix Anzar Rainbot was announcing every ~4 minutes even in an empty chat; now only speaks up when there are at least 3 recent messages and at most once per 30 minutes.
  • 2026-04-12 · ROFLFaucet — Deployment Bot deploy scripts rewritten to match the Python setup — previously broken and referencing old Node.js files.
  • 2026-04-11 · ROFLFaucet — Bot Migration Both Anzar Rainbot and ROFLBot migrated from Node.js to Python for improved stability and zero external dependencies. Bots now running on Orange Pi 5 via PM2 with 40% lower memory usage. Deployed and tested successfully.
  • 2026-04-09 · ROFLFaucet — Chat System Migrated chat system from WebSocket to HTTP polling API for improved reliability and scalability. Both bots updated to use new polling-based architecture.
